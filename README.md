# 💡 임베디드 시스템: CoAP 통신 실습
> **경험 요약:** CoAP 프로토콜의 구조를 이해하고, 센서 데이터를 효율적으로 전송하는 환경을 구축함.

---

## 🛠 사용 기술 및 환경
*   **Language:** C (Arduino), Python
*   **Protocol:** CoAP (Constrained Application Protocol)
*   **Hardware:** ESP32 / Arduino (실습 환경에 맞게 수정)

---

## 📋 주요 실습 내용
1. **CoAP 서버/클라이언트 구축:** 저전력 IoT 기기에 최적화된 통신 확인
2. **패킷 분석:** Wireshark 등을 활용한 메시지 구조 파악
3. **데이터 시각화:** 센서 데이터를 수집하여 리포트로 정리
